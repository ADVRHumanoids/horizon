horizon.solvers.sqp.sqpsol(name, qp_solver, problem_dict, options_dict)¶

sqpsol creates a sqp solver using Gauss-Newton approximation of the Hessian :param name: name of the solver (not used at the moment) :param qp_solver: internal qp solver name :param problem_dict: {‘f’: cost_function (residual!), ‘g’: constraints, ‘x’: variables} :param options_dict: {‘max_iter’: iterations}

Returns: a sqp object

NOTE: the cost function has to be passed in a residual form: [f1, f2, f3, …] where F = ||f1|| + ||f2|| + …
