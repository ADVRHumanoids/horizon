Horizon docker¶

Try Horizon on Docker!

Hint

if you don’t have Docker installed in your system, follow the steps in the official Docker installation page.


  1. clone the Horizon repo in your workspace.

  2. download the docker: docker pull francescoruscelli/horizon

  3. run the docker from your machine: ./horizon/docker/run.sh

  4. you are now inside the Horizon docker.
  5. From here, you can play around with the examples available. A list of possible commands:

    • python3 -m horizon.examples.spot --r --action walk
    • python3 -m horizon.examples.spot --r --action leap
    • python3 -m horizon.examples.cart_pole --r

  • visualize the robot on your browser on this static webpage ..warning:

    the static webpage is generated by the example you are running, which uses an implementation of rviz online to visualize the robot.
