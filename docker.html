Horizon docker¶

Try Horizon on Docker! .. container:: toggle

Docker

if you don’t have Docker installed in your system, follow the steps in the official Docker webpage.

  • download the Horizon repo
  • download the docker: docker pull francescoruscelli/horizon
  • run the docker from your machine: ./horizon/docker/run.sh
  • you are now inside the Horizon docker. You can play around with the examples available. A list of possible commands:
    1. ``python3 -m horizon.examples.spot –r –action walk
    2. ``python3 -m horizon.examples.spot –r –action leap
    3. ``python3 -m horizon.examples.cart_pole –r
    4. …
  • visualize the robot on your browser on this static ``webpage <http://localhost:8001/rvizweb/www/index.html>`_
